[testmag.ml](http:/testmag.ml) мой (практикум) интернет магазин на вольную тему построенный на "чистом" PHP (ООП) с использованием паттерна MVC. БД на MySQL. На сайте работает: регистрация(Register) и вход(Login) с валидацией, выход(Logout), кабинет юзера(Account), корзина и многое другое... Верстка с применением Bootstrap 3, JQuery. А также интерактив в виде: SimpleCart.js , MeMenu.js , ResponsiveSlides.js , FlexSlider.js
